# 文字处理节

页码

节中页面的页码使用<w:pgNumType>元素定义。

<w:sectPr>

<w:pgNumType w:start="25"/>

</w:sectPr>

参考：ECMA-376，第3版（2011年6月），基础和标记语言参考§ 17.6.12。

### 属性

| 元素    | 描述                                                                 |
| ------- | -------------------------------------------------------------------- |
| chapSep | 指定如果设置了章节样式，在章节和页码之间使用的分隔字符。可能的值有： |

- 冒号（例如，1:1）
- 长破折号 - （例如，1—1）
- 短破折号 - （例如，1–1）
- 连字符 - （例如，1-1
- 句点 - （例如，1.1）

chapStyle | 指定应用于章节标题的标题样式的从1开始的索引。定位最近的该样式的标题，并提取编号信息用于页码。例如，chapStyle值为1（标题1样式）意味着最近的标题1样式的编号值用于页码中的章节值。  
fmt | 指定节中页码使用的数字格式。最常用的值有：

- cardinalText - 运行语言的基本数词文本。（在英语中，One、Two、Three等）
- decimal - 十进制编号（1、2、3、4等）
- decimalEnclosedCircle - 用圆圈包围的十进制数字
- decimalEnclosedFullstop - 后跟句点的十进制数字
- decimalEnclosedParen - 用括号包围的十进制数字
- decimalZero - 十进制数字，但在1到9的数字前添加零
- lowerLetter - 基于运行语言（例如，a、b、c等）。对于大于字母表大小的值，字母会重复
- lowerRoman - 小写罗马数字（i、ii、iii、iv等）
- none
- ordinalText - 运行语言的序数文本。（在英语中，First、Second、Third等）
- upperLetter - 基于运行语言（例如，A、B、C等）。对于大于字母表大小的值，字母会重复
- upperRoman - 大写罗马数字（I、II、III、IV等）

start | 指定出现在节第一页上的页码。如果省略该值，编号将从前一节的最高页码继续。

### 相关HTML/CSS属性：

HTML没有页面的概念，因此也没有页码。
